## Changes

### Arrays -> Tables
Arrays are now functionally tables, default keys are doubles but allow for mixed key and value typing.
```c
  pin foo = [1, 2, 3];
  print(foo); // [1, 2, 3]

  pin bar = [1, "key": "value", 2];
  print(bar); // [0: 1, "key": "value",  1: 2]
```
### Classes now allow returning of `this`
```c
class Foo {
  pin x = 0;
  fn add(y) {
    this->x = this->x + y;
    return this;
  }
}

pin foo = new Foo();
foo.add(1);
print(foo.x); // 1
foo.add(1).add(2);

print(foo.x); // 4
```
### Module System
A basic module system has been added and supports custom dynamic libraries.
To use a module, simply `include` it without file extension.

```c
include "random";

seed(42);
print(choice(["red", "green", "blue"])); // green
```

### Install Script
Install scripts stores libraries in the following paths.

**Linux**
.sis libs: `~/.config/sis/lib/managed`
.so libs: `~/.config/sis/lib/dynamic`

**Windows**
.sis libs: `C:\Users\<name>\AppData\Local\SiS\lib\managed`
.dll libs: `C:\Users\<name>\AppData\Local\SiS\lib\dynamic`

The install scripts currently ship the following libraries
- io
- regex
- time
- os
- math
- random
- string
- vector

(Documentation support for individual functions coming soon. For now see the [source code](https://github.com/Julius-IX/SiS/tree/main/stdlib/dynamic), most are simple wrappers.)
