## Scoping Changes
Includes are no longer transitive and are now only valid in the current file.
Includes also now allow for aliasing via the same general syntax as Python imports.
Nested include access is not possible.

```C
include "foo.sis" as foo;

foo.bar();
```

## Explicit Documentation
Documenting named function and class behavior is now possible by accessing the `__docs__` variables present in functions and classes

```C
/// This Class does this
class Foo {

	/// This method says hello
  fn bar() {
		print("Hello World!");
	}
}

/// This free function does nothing
fn something() {}

pin instance = new Foo();
print(instance.__docs__);
print(instance.bar.__docs__);
print(something.__docs__);

```

### Cli Flags
A basic set of flags are now available for the SiS interpreter, and function as any normal interpreter.
```fish
usage: sis [options] [file]
  <file>              run script
  -e <code>           evaluate string
  --interactive / -i  enter REPL after script (or alone)
  --help    / -h      show this message
  --version / -v      show version
  --parallel / -p     use parallel lexing
```
